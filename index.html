Raze Closed

Raze was a military-political Minecraft server running version 1.20.1, created and released to the public in December 2025. Developed by myself, the entire server logic was written in Java and compiled into a single mod. The server featured a single PvP mode, Annihilation, with the destruction of military aircraft control points. The project was closed in January 2026 due to financial inability to maintain the server.

ecco Closed

ecco is a solo experimental project created in February 2026 by myself. It is a universal Telegram bot for downloading high-quality media content from various platforms, such as YouTube, TikTok, Instagram, Pinterest, and SoundCloud. The project is written in Python and uses the yt-dlp library. Downloading from YouTube was suspended after a week due to accessibility issues. The project was closed due to its lack of relevance and the release of the server in favor of other projects.

H.S.B Active

H.S.B (Halyava Seeker Bot) is a Telegram bot created in 2026 by myself for discovering free and discounted games across multiple platforms, including Steam, Epic Games Store, GamerPower, and itch.io. Written in Python using the aiogram framework, the bot features automated giveaway tracking, new release notifications filtered by genre preferences, and a personal wishlist system with price drop alerts. The bot supports multiple languages and currencies, with scheduled background checks running every 30 minutes to 6 hours depending on the data source. The bot is available on Telegram under the username @HSB_Pidorbot

SixteenCloud Active

SixteenCloud is a custom, unofficial SoundCloud fork with removed ads, regional restriction bypass, Discord Rich Presence, OAuth Token Login, a built-in track downloader, and many useful features. Created by me in April 2026, it's now live. Download it here.
